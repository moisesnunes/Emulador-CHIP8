# Emulador-CHIP8
Emulador do CHIP8 com SDL/openGL e ImGUI

![chip8](./Img/chip8.gif)

> E necessário que seja adcionado o imgui no diretório pois é usado o cimgui.
O Emulador já vem com algumas roms para fazer testes dentro da pasta __Debug/roms__
> O executável também é enviado para está pasta

> Digite __make__ para compilar.

> Desenvolvido e Testado apenas no Linux.
